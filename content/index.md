# Welcome to Shizuka

This is a test page for the Shizuka static site generator.

## Features

- Fast builds with Go
- Markdown content
- Template-based rendering
- Live reload development server

## Getting Started

1. Create content in the `content/` directory
2. Add templates to `templates/`
3. Run `shizuka dev` to start the development server
4. Run `shizuka build` to build your site

Enjoy building with Shizuka!
